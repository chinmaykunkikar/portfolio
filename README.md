### Local setup

1. Clone this repository.
2. Install dependencies using `npm` or [`bun`](https://bun.sh/):

```bash
bun install
```

3. Start the development server:

```bash
bun run dev
```

Then open **[localhost:3000](http://localhost:3000)** on your favorite browser.
